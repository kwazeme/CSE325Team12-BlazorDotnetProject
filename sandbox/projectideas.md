## 1. Community Event Management Platform

**Target Audience:** Local community groups, NGOs, churches, student organizations

**Description:**
A web application that allows organizers to create events and community members to discover, register, and manage their attendance.

**Core Features:**

* User authentication (Organizer vs Attendee roles)
* CRUD:
  * Events (create, edit, delete, view)
  * Event registrations
  * Event categories
* Event calendar and search/filter
* Email confirmation for registrations
* Dashboard for organizers

**Why It Fits Perfectly:**

* CRUD is central to the app
* Clear role-based authentication
* Performance focus (lazy loading event lists)
* Accessibility: keyboard navigation, ARIA labels
* Easy to deploy on Azure App Service
* Strong usability & branding potential
